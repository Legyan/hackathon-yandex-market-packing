## Запуск frontend:
- npm i,
- npm run start

## Реализованные фичи: 
- Авторизация,
- Роутинг,
- Модальные окна, 
- Получение заказа,
- Выбор рекомендации упаковки,
- Контроль выбора упаковки,
- Контроль выбора товара:
- Фиксация штрих кода, IMEI, Честный знак,
- Фиксация возникшей проблемы,

## Используемые технологии:
- TypeScript,
- React,
- Redux,
- React Router,
- flexbox верстка,
- Css modules,
- REST API,
